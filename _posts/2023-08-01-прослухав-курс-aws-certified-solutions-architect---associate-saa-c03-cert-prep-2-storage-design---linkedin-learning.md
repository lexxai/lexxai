### Course details

2h 35m  
Intermediate  
Released: 4/12/2023

In this course, learn the fundamentals of AWS storage design. Instructor
Tom Carpenter
begins with an overview of AWS storage services, then dives into deeper
discussion on Amazon S3 specifics. Additional storage services covered
include Glacier, Elastic Block Store, and Elastic File System. This
course provides foundational instruction for anyone beginning their
journey as an AWS architect, and also provides essential instruction for
anyone preparing for the AWS Solutions Architect – Associate exam.
